# BECOMING A MASTER

Master represents the highest validated stage of class understanding and character progression.

## Core Requirements

- Reach **115 Mastery Marks**.
- Submit a Master application (roleplay route or DM event route).
- Demonstrate strong class identity through mechanics and narrative behavior.

Master approval is not automatic and is not intended as a pure power milestone. It is a roleplay and competency benchmark.

## Expectations for Master Characters

Master characters are held to elevated standards in both roleplay and conduct.

- In-character: uphold class-defining principles, themes, and obligations.
- Out-of-character: model strong rules understanding, play-to-lift behavior, and non-toxic conduct.

Failure to meet expectations can result in:

- reduction from Master to Expert, or
- full progression reset in serious cases.

Players can re-apply later, but must re-complete required progression.

## Route 1: Mastery Through DM Event

Use this route when an Expert-level character at high marks completes an event a DM agrees demonstrates mastery.

Process:

1. Inform the DM of your mark status during/after the event.
2. Request event sponsorship for mastery qualification.
3. Submit application summary explaining how class mastery was demonstrated.
4. Include DM sponsorship / co-sign context where applicable.

DM sponsorship is discretionary.

## Route 2: Mastery Through Roleplay Application

Use this route when your character arc supports mastery through sustained roleplay progression.

Typical application structure includes:

- Character name, class, and subclass.
- Three major prior roleplay events tied to class development.
- Qualities that make the character master-level.
- How the character channels and advances class power (mechanical and narrative).
- How subclass specialization is represented in play.
- How you intend to play-to-lift as a master.
- Any past disciplinary context and resolution.
- Acknowledgement that mastery can be suspended/revoked.

Concise, specific, and class-grounded submissions are reviewed more effectively than broad or generic narratives.
